# Multi-Paradigm Learning for Medicinal Plant Recognition

## Overview
This project focuses on medicinal plant recognition using multiple deep learning paradigms including CNN, YOLO, and Graph Neural Networks (GNN). The goal is to benchmark different architectures on a balanced dataset and analyze their performance for classification and detection tasks.

This work has been accepted at **IEEE QPAIN 2026**.

## Key Features
- CNN-based classification model
- YOLO-based object detection
- Graph Neural Network exploration
- Unified reproducible ML pipeline
- Performance benchmarking across models

## Dataset
A balanced medicinal plant dataset was used for training and evaluation.

## Results
Achieved near-perfect accuracy across selected architectures on validation data.

## Tech Stack
- Python
- PyTorch / TensorFlow
- OpenCV
- YOLO framework
- NetworkX (for GNN)
